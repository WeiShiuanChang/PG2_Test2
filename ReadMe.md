# 操作方法

* スタート：[ENTER]キー

* 移動：
    * 上：[W]キー または [UP]キー
    * 下：[S]キー または [DOWN]キー
    * 左：[A]キー または [LEFT]キー
    * 右：[D]キー または [RIGHT]キー

* 発射：[SPACE]キー

* タイトルページに戻る：[R]キー
